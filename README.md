# MacrosCoach API (FastAPI)

Backend del progetto **MacrosCoach**.

## Requisiti
- Python 3.10+ (WSL Ubuntu)
- `pip`, `venv`
- (Opzionale) PostgreSQL

## Setup rapido
```bash
cd ~/apps/macroscoach/api
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
